## 📌 Project Overview
This project is created as part of the Digital Literacy course. The objective is to understand and apply essential digital skills required in academic and professional environments. It includes creating awareness about digital literacy, building an online portfolio, exploring digital platforms, practicing professional communication, and understanding cybercrime prevention.

Created an infographic using Canva explaining digital literacy, useful tools, safe internet practices, professional presence, and email etiquette.

### 🔹 Task 2: Digital Portfolio
Set up professional profiles on platforms like GitHub and LinkedIn to build a digital presence.

### 🔹 Task 3: Platforms Exploration
Explored coding platforms and created a Google Form for digital literacy awareness quiz.

### 🔹 Task 4: Email Etiquette
Drafted professional emails and created a checklist for responsible social media usage.

### 🔹 Task 5: Cybercrime Awareness
Prepared a case study on cybercrime and a prevention checklist to promote safe online practices.
